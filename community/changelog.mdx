## Latest Stable Release

The latest stable version of TorchGeo can be installed via pip:

```bash
pip install torchgeo
```

For conda installation:

```bash
conda install -c conda-forge torchgeo
```

For spack installation:

```bash
spack install py-torchgeo
```

## Development Version

To install the latest development version with all features:

```bash
git clone https://github.com/torchgeo/torchgeo.git
cd torchgeo
pip install -e ".[all]"
```

## Version History

TorchGeo has evolved significantly since its initial release. Major milestones include:

- **Pre-trained weights** for multiple satellite sensors
- **Lightning datamodules and trainers** for reproducible research
- Support for **geospatial datasets** with coordinate reference systems
- **100+ datasets** for various remote sensing tasks
- **Multiple computer vision tasks**: classification, segmentation, detection, regression, and more

For a complete history of changes, see the [releases page](https://github.com/torchgeo/torchgeo/releases).

## Deprecation Policy

TorchGeo follows these deprecation guidelines:

- Deprecated features are marked with `.. deprecated::` in docstrings
- Deprecation warnings are raised when deprecated features are used
- Deprecated features are typically removed after 2 minor version releases
- Breaking changes are documented in release notes

## Release Schedule

TorchGeo does not follow a fixed release schedule. New versions are released when:

- Significant new features are added
- Critical bugs are fixed
- Breaking changes need to be introduced
