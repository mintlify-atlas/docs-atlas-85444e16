TorchGeo trainers (tasks) are PyTorch Lightning modules that define the training loop, loss functions, and metrics for various geospatial tasks.

## Base Task

### BaseTask

Abstract base class for all TorchGeo trainers.

<ParamField path="ignore" type="Sequence[str] | str | None" default="'weights'">
  Parameters to ignore when saving hyperparameters
</ParamField>

**Key Methods:**
- `configure_models()` - Initialize the model (abstract)
- `configure_losses()` - Initialize the loss criterion
- `configure_metrics()` - Initialize performance metrics
- `configure_optimizers()` - Initialize optimizer and LR scheduler (default: AdamW + ReduceLROnPlateau)

## Complete Training Example

```python
from lightning.pytorch import Trainer
from lightning.pytorch.callbacks import ModelCheckpoint, EarlyStopping
from torchgeo.datamodules import EuroSATDataModule
from torchgeo.trainers import ClassificationTask

# Initialize datamodule
datamodule = EuroSATDataModule(
    root='data',
    batch_size=128,
    num_workers=6,
    download=True
)

# Initialize task
task = ClassificationTask(
    model='resnet50',
    in_channels=13,  # Sentinel-2 has 13 bands
    num_classes=10,
    loss='ce',
    lr=1e-3,
    patience=5,
    freeze_backbone=False
)

# Callbacks
checkpoint_callback = ModelCheckpoint(
    monitor='val_loss',
    mode='min',
    save_top_k=1,
    filename='best-{epoch:02d}-{val_loss:.2f}'
)

early_stopping = EarlyStopping(
    monitor='val_loss',
    patience=10,
    mode='min'
)

# Trainer
trainer = Trainer(
    max_epochs=100,
    callbacks=[checkpoint_callback, early_stopping],
    accelerator='gpu',
    devices=1,
    precision='16-mixed'
)

# Train
trainer.fit(model=task, datamodule=datamodule)

# Test
trainer.test(model=task, datamodule=datamodule, ckpt_path='best')
```

## Custom Tasks

You can create custom tasks by subclassing `BaseTask`:

```python
from torchgeo.trainers import BaseTask
import torch.nn as nn

class CustomTask(BaseTask):
    def __init__(self, num_classes=10, lr=1e-3):
        self.num_classes = num_classes
        self.lr = lr
        super().__init__()
    
    def configure_models(self):
        """Initialize your model."""
        self.model = YourCustomModel(
            num_classes=self.num_classes
        )
    
    def configure_losses(self):
        """Initialize loss function."""
        self.criterion = nn.CrossEntropyLoss()
    
    def training_step(self, batch, batch_idx):
        """Training step."""
        x = batch['image']
        y = batch['label']
        y_hat = self(x)
        loss = self.criterion(y_hat, y)
        self.log('train_loss', loss)
        return loss
    
    def validation_step(self, batch, batch_idx):
        """Validation step."""
        x = batch['image']
        y = batch['label']
        y_hat = self(x)
        loss = self.criterion(y_hat, y)
        self.log('val_loss', loss)
        return loss
```

## Key Features

- **Automatic optimization**: Built-in AdamW + ReduceLROnPlateau
- **Hyperparameter logging**: All parameters saved via `save_hyperparameters`
- **Multiple losses**: Support for CE, BCE, Jaccard, Focal, Dice
- **Flexible architectures**: Support for timm models, torchvision models, custom models
- **Transfer learning**: Load pre-trained weights and optionally freeze backbone
- **Multi-channel support**: Handle arbitrary number of input channels
