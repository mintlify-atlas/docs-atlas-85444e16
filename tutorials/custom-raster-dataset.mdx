TorchGeo's `RasterDataset` class makes it easy to create custom datasets from your own geospatial imagery. This tutorial shows you how to build a dataset that automatically indexes and loads your raster files.

## Overview

`RasterDataset` is an abstract base class that:
- Automatically discovers files matching a pattern
- Extracts metadata (bounds, CRS, resolution)
- Parses timestamps from filenames
- Handles reprojection and resampling
- Supports both single-file and multi-file bands

<Step title="Create a custom dataset class">

Subclass `RasterDataset` and define the filename patterns:

```python
import os
import re
from torchgeo.datasets import RasterDataset

class Sentinel2Dataset(RasterDataset):
    """Custom Sentinel-2 dataset."""

    # Glob pattern to find files
    filename_glob = "S2*.tif"

    # Regex to parse filename components
    filename_regex = r"""
        ^S2[AB]_
        (?P<date>\d{8})_  # Date: YYYYMMDD
        (?P<band>B\d{2})  # Band: B01, B02, etc.
        \.tif$
    """

    # Date format for parsing the 'date' group
    date_format = "%Y%m%d"

    # This dataset contains imagery (not labels)
    is_image = True

    # Each band is in a separate file
    separate_files = True

    # Available bands
    all_bands = ("B01", "B02", "B03", "B04", "B05", "B06",
                 "B07", "B08", "B8A", "B09", "B10", "B11", "B12")

    # RGB bands for visualization
    rgb_bands = ("B04", "B03", "B02")
```

</Step>

<Step title="Use the dataset">

Now you can use your dataset like any other TorchGeo dataset:

```python
# Create dataset instance
dataset = Sentinel2Dataset(
    paths="data/sentinel2",
    bands=["B04", "B03", "B02"],  # RGB bands
    crs="EPSG:32615",              # Target CRS
    res=10.0                        # 10m resolution
)

print(f"Found {len(dataset)} files")
print(f"Bounds: {dataset.bounds}")
print(f"CRS: {dataset.crs}")

# Index by geospatial coordinates
from datetime import datetime

sample = dataset[
    500000:500256:10,     # x: 256 pixels at 10m resolution
    4200000:4200256:10,   # y: 256 pixels at 10m resolution  
    datetime(2020, 1, 1):datetime(2020, 1, 31):  # January 2020
]

print(f"Image shape: {sample['image'].shape}")  # [3, 256, 256]
```

</Step>

```python
class Sentinel2SingleFile(RasterDataset):
    """Sentinel-2 with all bands in one file."""

    filename_glob = "S2*.tif"
    filename_regex = r"^S2[AB]_(?P<date>\d{8})\.tif$"
    date_format = "%Y%m%d"
    is_image = True

    # All bands in one file
    separate_files = False

    all_bands = ("B01", "B02", "B03", "B04", "B05", "B06",
                 "B07", "B08", "B8A", "B09", "B10", "B11", "B12")
    rgb_bands = ("B04", "B03", "B02")
```

<Step title="Label dataset example">

For segmentation masks or labels, set `is_image=False`:

```python
class LandCoverMasks(RasterDataset):
    """Land cover classification masks."""

    filename_glob = "landcover_*.tif"
    filename_regex = r"^landcover_(?P<date>\d{8})\.tif$"
    date_format = "%Y%m%d"

    # This is a mask/label dataset
    is_image = False

    # Color map for visualization (optional)
    cmap = {
        0: (0, 0, 0, 255),       # Background
        1: (0, 255, 0, 255),     # Forest
        2: (255, 255, 0, 255),   # Cropland
        3: (0, 0, 255, 255),     # Water
    }
```

The dataset will return samples with a `mask` key instead of `image`:

```python
masks = LandCoverMasks(paths="data/landcover")
sample = masks[500000:500256:10, 4200000:4200256:10, :]

print(f"Mask shape: {sample['mask'].shape}")  # [256, 256]
print(f"Mask dtype: {sample['mask'].dtype}")  # torch.long
```

</Step>

<Step title="Advanced: Date ranges">

For files covering a time range, use `start` and `stop` groups:

```python
class MonthlyComposite(RasterDataset):
    """Monthly composite imagery."""

    filename_glob = "composite_*.tif"

    # Capture start and end dates
    filename_regex = r"""
        ^composite_
        (?P<start>\d{8})_  # Start: YYYYMMDD
        (?P<stop>\d{8})    # End: YYYYMMDD
        \.tif$
    """

    date_format = "%Y%m%d"
    is_image = True
```

This indexes each file with a time interval instead of a single timestamp.

</Step>

</Steps>

## Key Attributes

### Required Attributes

| Attribute | Description | Example |
|-----------|-------------|----------|
| `filename_glob` | Pattern to find files | `"*.tif"`, `"L8*.TIF"` |
| `filename_regex` | Regex to parse filenames | See examples above |
| `is_image` | True for imagery, False for masks | `True` |

### Optional Attributes

| Attribute | Description | Default |
|-----------|-------------|---------|
| `date_format` | Format string for date parsing | `"%Y%m%d"` |
| `separate_files` | True if each band is a separate file | `False` |
| `all_bands` | Tuple of available band names | `()` |
| `rgb_bands` | Tuple of RGB band names for plotting | `()` |
| `cmap` | Color map for masks | `{}` |

### Regex Named Groups

The `filename_regex` can include these special groups:

- `date`: Single timestamp (uses `date_format`)
- `start`: Start of time range (uses `date_format`)
- `stop`: End of time range (uses `date_format`)
- `band`: Band name (only when `separate_files=True`)

## Combining Datasets

Combine your custom dataset with labels:

```python
# Load imagery and masks
imagery = Sentinel2Dataset(paths="data/imagery")
masks = LandCoverMasks(paths="data/masks")

# Intersect them
dataset = imagery & masks

# Sample from both
from torchgeo.samplers import RandomGeoSampler
from torch.utils.data import DataLoader

sampler = RandomGeoSampler(dataset, size=256, length=10000)
dataloader = DataLoader(dataset, sampler=sampler, collate_fn=stack_samples)

for batch in dataloader:
    images = batch["image"]  # From Sentinel2Dataset
    masks = batch["mask"]    # From LandCoverMasks
```

## Time Series Support

Enable time series mode to stack temporal data:

```python
dataset = Sentinel2Dataset(
    paths="data/sentinel2",
    time_series=True  # Stack along time dimension
)

# Query a time range
sample = dataset[
    500000:500256:10,
    4200000:4200256:10,
    datetime(2020, 1, 1):datetime(2020, 12, 31):
]

print(f"Shape: {sample['image'].shape}")  # [T, C, H, W]
```

Without `time_series=True`, multiple timestamps are merged into a mosaic instead.
