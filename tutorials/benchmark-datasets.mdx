TorchGeo includes numerous benchmark datasets for common remote sensing tasks like classification, segmentation, and object detection. These datasets have pre-defined train/val/test splits and work similarly to torchvision datasets.

## Overview

Benchmark datasets (also called "NonGeoDatasets") contain pre-defined image chips with corresponding labels. They're ideal for:

- Reproducible research with standard splits
- Direct comparison with published results
- Quick experimentation without geospatial complexity

## Complete Example with EuroSAT

<Steps>

<Step title="Import required modules">

```python
import torch
import torch.nn as nn
from torch.utils.data import DataLoader
from torchgeo.datasets import EuroSAT
import timm
```

</Step>

<Step title="Initialize the dataset">

EuroSAT is a Sentinel-2 classification dataset with 10 land cover classes:

```python
# Training set
train_dataset = EuroSAT(
    root="data/eurosat",
    split="train",
    download=True,   # Automatically download
    checksum=True    # Verify integrity
)

# Validation set
val_dataset = EuroSAT(
    root="data/eurosat",
    split="val",
    download=True,
    checksum=True
)

print(f"Train samples: {len(train_dataset)}")
print(f"Val samples: {len(val_dataset)}")
print(f"Classes: {train_dataset.classes}")
```

Classes in EuroSAT:
- AnnualCrop
- Forest
- HerbaceousVegetation
- Highway
- Industrial
- Pasture
- PermanentCrop
- Residential
- River
- SeaLake

</Step>

<Step title="Inspect a sample">

Each sample is a dictionary with `image` and `label` keys:

```python
sample = train_dataset[0]
print(f"Image shape: {sample['image'].shape}")  # [13, 64, 64]
print(f"Label: {sample['label']}")              # Tensor (class index)
print(f"Class name: {train_dataset.classes[sample['label']]}")

# EuroSAT has 13 spectral bands from Sentinel-2
```

</Step>

<Step title="Create dataloaders">

```python
train_loader = DataLoader(
    train_dataset,
    batch_size=64,
    shuffle=True,
    num_workers=4,
    pin_memory=True
)

val_loader = DataLoader(
    val_dataset,
    batch_size=64,
    shuffle=False,
    num_workers=4,
    pin_memory=True
)
```

</Step>

<Step title="Create a model">

Use timm to create a ResNet-18 model for 13-channel input:

```python
model = timm.create_model(
    "resnet18",
    in_chans=13,      # EuroSAT has 13 spectral bands
    num_classes=10    # 10 land cover classes
)

model = model.cuda()
criterion = nn.CrossEntropyLoss()
optimizer = torch.optim.Adam(model.parameters(), lr=1e-3)
```

</Step>

<Step title="Training loop">

```python
def train_epoch(model, dataloader, criterion, optimizer, device):
    model.train()
    total_loss = 0
    correct = 0
    total = 0

    for batch in dataloader:
        images = batch["image"].to(device)
        labels = batch["label"].to(device)

        optimizer.zero_grad()
        outputs = model(images)
        loss = criterion(outputs, labels)
        loss.backward()
        optimizer.step()

        total_loss += loss.item()
        _, predicted = outputs.max(1)
        total += labels.size(0)
        correct += predicted.eq(labels).sum().item()

    accuracy = 100.0 * correct / total
    return total_loss / len(dataloader), accuracy

# Train for multiple epochs
for epoch in range(20):
    train_loss, train_acc = train_epoch(
        model, train_loader, criterion, optimizer, "cuda"
    )
    print(f"Epoch {epoch}: Loss={train_loss:.4f}, Acc={train_acc:.2f}%")
```

</Step>

<Step title="Evaluation">

```python
@torch.no_grad()
def evaluate(model, dataloader, device):
    model.eval()
    correct = 0
    total = 0

    for batch in dataloader:
        images = batch["image"].to(device)
        labels = batch["label"].to(device)

        outputs = model(images)
        _, predicted = outputs.max(1)

        total += labels.size(0)
        correct += predicted.eq(labels).sum().item()

    accuracy = 100.0 * correct / total
    return accuracy

val_acc = evaluate(model, val_loader, "cuda")
print(f"Validation Accuracy: {val_acc:.2f}%")
```

</Step>

</Steps>

## Available Datasets

TorchGeo includes many benchmark datasets:

### Classification
- **EuroSAT**: Sentinel-2 land cover classification (10 classes)
- **RESISC45**: RGB scene classification (45 classes)
- **UC Merced**: RGB land use classification (21 classes)
- **BigEarthNet**: Large-scale Sentinel-2 multi-label classification

### Semantic Segmentation
- **Inria Aerial**: Building segmentation from aerial imagery
- **LoveDA**: Urban/rural land cover segmentation
- **DeepGlobe Land Cover**: Land cover segmentation challenge
- **Potsdam**: Urban semantic segmentation

### Object Detection
- **VHR-10**: 10-class geospatial object detection
- **xBD**: Building damage assessment
- **COWC**: Car detection in overhead imagery

### Change Detection
- **OSCD**: Onera Satellite Change Detection
- **LEVIR-CD**: Building change detection

## Custom Transforms

Apply data augmentation with transforms:

```python
import torchvision.transforms.v2 as T

def transform(sample):
    # Normalize and augment
    sample["image"] = T.Normalize(
        mean=[0.485, 0.456, 0.406],
        std=[0.229, 0.224, 0.225]
    )(sample["image"])

    return sample

dataset = EuroSAT(
    root="data",
    split="train",
    transforms=transform
)
```
