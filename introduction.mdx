# Introduction to TorchGeo

TorchGeo is a PyTorch domain library, similar to torchvision, providing datasets, samplers, transforms, and pre-trained models specific to geospatial data.

<img src="https://raw.githubusercontent.com/torchgeo/torchgeo/main/docs/_static/logo/logo-color.svg" width="400" alt="TorchGeo logo" />

## Our Mission

The goal of TorchGeo is to make it simple:

1. **For machine learning experts** to work with geospatial data
2. **For remote sensing experts** to explore machine learning solutions

Whether you're a deep learning researcher looking to apply your models to satellite imagery, or a remote sensing scientist wanting to leverage the latest ML techniques, TorchGeo bridges the gap between these two domains.

## Key Features

### Geospatial Datasets

Work with datasets that have geographic metadata - different CRS, resolutions, and spectral bands. TorchGeo handles the complexity of reprojecting and aligning data automatically.

<img src="https://raw.githubusercontent.com/torchgeo/torchgeo/main/docs/_static/images/geodataset.png" alt="Example application combining Landsat and CDL" />

**Features:**
- Automatic CRS alignment and reprojection
- Multi-spectral and multi-resolution support
- Geospatial samplers for coordinate-based sampling
- Union and intersection operations for combining datasets

### Benchmark Datasets

Access 50+ benchmark datasets for various tasks:
- Image classification
- Semantic segmentation
- Object detection
- Instance segmentation
- Change detection
- Regression

All datasets support automatic download and checksum verification, just like torchvision.

### Pre-trained Models

<Note>
TorchGeo is the first library to support models pre-trained on multispectral satellite imagery!
</Note>

Unlike ImageNet pre-training which only covers RGB channels, TorchGeo provides weights trained on:
- Sentinel-2 (13 spectral bands)
- Landsat (7-11 bands depending on satellite)
- NAIP aerial imagery
- And more!

### Lightning Integration

Built-in PyTorch Lightning datamodules and trainers provide:
- Standard train/val/test splits for reproducibility
- Pre-configured augmentations using kornia
- Pre-calculated channel statistics
- Task-specific trainers (classification, segmentation, etc.)

## Why TorchGeo?

### For ML Experts

If you're familiar with PyTorch and torchvision but new to geospatial data, TorchGeo abstracts away the complexity:
- No need to learn GDAL, rasterio, or geospatial concepts
- Familiar PyTorch Dataset and DataLoader interfaces
- Pre-built models and training pipelines
- Benchmark datasets to test your ideas

### For Remote Sensing Experts

If you work with satellite or aerial imagery but are new to deep learning:
- Easy integration with existing geospatial workflows
- Automatic handling of CRS, resolution, and alignment
- Pre-trained models specialized for remote sensing
- Reproducible research with standard benchmarks
